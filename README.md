# வெண்முரசு (Venmurasu Reader)

எழுத்தாளர் ஜெயமோகனின் "வெண்முரசு" நாவல் தொடரை வாசிப்பதற்கான ஒரு நவீன இணையச் செயலி.

## முக்கிய வசதிகள் (Features)

- **நூல்கள் & அத்தியாயங்கள்**: வெண்முரசின் அனைத்து நூல்களையும் அத்தியாய வாரியாக வாசிக்கலாம்.
- **உறவுகள் (Family Trees)**: கதாபாத்திரங்களுக்கு இடையிலான உறவுமுறைகளை வரைபடமாகப் பார்க்கலாம்.
- **கலைச்சொற்கள் (Glossary)**: நாவலில் வரும் கடினமான சொற்களுக்கான விளக்கங்கள்.
- **வரைபடம் (Map)**: வரலாற்று இடங்களை வரைபடம் மூலம் கண்டறியலாம்.
- **அடையாளங்கள் (Bookmarks)**: பிடித்த பகுதிகளைச் சேமிக்கலாம்.
- **நிர்வாகப் பகுதி (Admin)**: நாவல் தரவுகளை சுலபமாக நிர்வகிக்கலாம்.

## தொழில்நுட்பம் (Tech Stack)

- **Frontend**: React, TypeScript, Vite, Tailwind CSS
- **UI Components**: Shadcn UI, Lucide Icons
- **Backend**: Supabase (Database & Auth)
- **Data Fetching**: TanStack Query (React Query)

## உள்ளூர் அமைவு (Local Setup)

இந்தச் செயலியை உங்கள் கணினியில் இயக்க:

1. **கோப்புகளைப் பதிவிறக்கவும்** (Clone/Download).
2. **Dependencies நிறுவவும்**:
   ```bash
   npm install
   ```
3. **Environment Variables**: `.env` கோப்பில் உங்கள் Supabase URL மற்றும் Key உள்ளதை உறுதி செய்யவும்.
4. **இயக்கவும் (Run)**:
   ```bash
   npm run dev
   ```
   இப்போது `http://localhost:8080`-ல் ஆப் இயங்கும்.

5. **பில்ட் (Build)**:
   ```bash
   npm run build
   ```
   இது `dist` என்ற கோப்புறையை (folder) உருவாக்கும். இதை நீங்கள் எந்த இணைய முகவரியிலும் பதிவேற்றலாம்.
